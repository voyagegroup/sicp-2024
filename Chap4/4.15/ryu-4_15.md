> 一引数手続きpとオブジェクトaが与えられ, 式(p a)の評価が, (エラーメッセージで停止したり, 永遠に走ったりせず)値を返すようなら, pはaで「停止する」という.

### 停止するとは

これは停止する

```
(define (p a) (display a))
(p 'hello)
```

これはエラーなので停止しない

```
(define (p a) (error 'error'))
(p 'hello)
```

これは無限にhelloが続くので停止しない

```
(define (p a) (display a (p a)))
(p 'hello)
```

### サンプルコード

```
; 無限ループ
(define (run-forever) (run-forever))

(define (try p)
  (if (halts? p p)
      (run-forever) ; 停止したら無限ループ
      'halted)) ; 停止しないなら、'haltedを返す
```

### (try try) を実行するとどうなるか

1. (halts? try try) → #t とする

- try は try で停止すると判定された
- try が停止されるということは、(run-forever)が実行されるということ
- ということは、ループに陥っているので、停止しないということ
- halts? の判定と矛盾が生じる


2. (halts? try try) → #f とする

- try は try で停止しないと判定された
- try が停止されないということは、 'halted が帰る
- ということは、処理が停止されるということ
- halts? の判定と矛盾が生じる
