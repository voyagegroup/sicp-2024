# 問題 4.62

リストの最後のペアを見つける `last-pair` ルールを定義する。

## ルール定義

```scheme
(rule (last-pair (?x) (?x)))

(rule (last-pair (?v . ?w) ?x)
      (last-pair ?w ?x))
```

1つ目: 要素が1つのリストの最後のペアはそれ自身。  
2つ目: 先頭をスキップして残りで再帰。

---

## クエリと結果

```scheme
(last-pair (3) ?x)
; => ?x = (3)

(last-pair (1 2 3) ?x)
; => ?x = (3)

(last-pair (2 ?x) (3))
; => ?x = 3
```

`(last-pair (2 ?x) (3))` の導出：
- ルール2で `?v=2`, `?w=(?x)` → `(last-pair (?x) (3))`
- ルール1で `(?x) = (3)` → `?x = 3`

---

## `(last-pair ?x (3))` は正しく動くか

最初の答えは返ってくる：
- ルール1: `?x = (3)` ✓

しかしルール2も試みる：
- `?x = (?v . ?w)` として `(last-pair ?w (3))` を再帰 → `?w = (3)` → `?x = (?v 3)`（`?v` は自由変数）
- さらにルール2を再帰 → `?x = (?v1 ?v2 3)`（`?v1`, `?v2` は自由変数）
- 無限に長いリストを生成し続ける

最初の解 `(3)` は正しいが、その後**無限ループ**になるため正しく動くとは言えない。
